<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added natural language querying with AI-powered insights. Ask questions like "Show top revenue sources last quarter" directly in the interface.
- Introduced real-time collaboration on reports. Multiple team members can edit and comment simultaneously.

## Improvements

- Enhanced data source integrations with Google Analytics and PostgreSQL, reducing setup time by 40%.
- Optimized dashboard rendering for datasets `{>1M rows}`.

## Bug Fixes

- Fixed intermittent crashes when exporting large CSV files.
- Resolved sync issues with Snowflake connectors.

```javascript
// Example: New AI query API usage
const response = await fetch('https://api.example.com/v1/query/ai', {
  method: 'POST',
  headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
  body: JSON.stringify({ query: 'top revenue sources Q3' })
});
```

</Update>

## How to Upgrade

<Steps>
  <Step title="Check Current Version" icon="check-circle">

Visit your dashboard at `https://dashboard.example.com/settings` and note the version.

  </Step>
  <Step title="Review Changes" icon="book-open">

Read the release notes above and test in staging.

  </Step>
  <Step title="Update SDK" icon="download">
    <CodeGroup tabs="npm,yarn">
````bash
npm update @baza/analytics-sdk@latest
````
````bash
yarn upgrade @baza/analytics-sdk@latest
````
    </CodeGroup>
  </Step>
  <Step title="Restart Services" icon="refresh-cw">

Restart your app and verify queries work.

  </Step>
</Steps>
